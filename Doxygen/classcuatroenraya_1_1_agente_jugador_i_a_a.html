void cuatroenraya.AgenteJugadorIAA.setup ( )
protected

Inicialización del agente donde se prepara la ontología y se registra en las páguinas amarillas, lanza la Tarea de GanadorJuego para saber cuando ha ganado, también lanza una tarea encargada de reiniciar el AgenteJugador en caso de que esté mucho tiempo esperando respuesta. Después de que todo esto este hecho, llama a la tarea IniciarJuego

Obtenemos la instancia de la ontología y registramos el lenguaje y la ontología para poder completar el contenido de los mensajes

Registro del agente en las páginas amarillas

Agents that want to use this service need to "know" the weather-forecast-ontology

Agents that want to use this service need to "speak" the FIPA-SL language

Tarea que nos informa el ganador

Plantilla la exploración del mensaje para el protocolo ContractNet
